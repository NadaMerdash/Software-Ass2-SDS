# Software-Ass2-SDS
## Software Design Specification Assignment:
  By doing this assignment, you will:
  
    1- Learn about opportunities for computing students and possible paths for improvement.
    
    2- Learn about SW design. Move from SRS (what) to design (how) to implementation (real SW)
    
    3- Learn how to develop architecture, class, state and sequence diagrams to describe a system.
    
    4- Learn how to enhance designs / code with SOLID principles and design patterns.
    
    5- Learn how to transform design models to code.
    
    6- Learn how to document code and to use GitHub as a repository for the project.
